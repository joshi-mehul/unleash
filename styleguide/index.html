X-Team Unleash App CSS Styleguide

This document defines the CSS classnames to be used in the Unleash App redesign, establishing patterns that everybody can follow.

Basic formatting definitions:

  • Using BEM as naming convention, as it allows to treat each design block as a module.
  • A single CSS file per module. (eg. header.js -> header.css, profile-recap.js -> profile-recap.css, etc)
  • Using dashes over camelCasing in classnames. (eg. user-settings instead of userSettings)
